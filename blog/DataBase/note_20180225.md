# Introduction to DataBase

_25 Feb 2018_ (补档)

## Concepts

### DataBase

* 长期存储在计算机中的、有组织的、可共享的数据集合
* 特点
  * 按照一定的数据模型组织、描述和存储
  * 具有较小的冗余度
  * 具有较高的数据独立性和易拓展性
  * 可为各种用户共享

### DataBase Management System(DBMS)

* 是数据管理软件
* 用于建立、运用和维护数据库
* 位于用户和操作系统之间

### DataBase System(DBS)

* 是指计算机系统中引入数据库后的系统而构成，一般包括:
  * DB
  * DBMS+OS
  * Users (optional)
  * DBA
  * 应用系统
